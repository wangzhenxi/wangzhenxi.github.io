# 清单革命

# 简介

清单革命，为了解决错误。将靠记忆解决的事情，用清单解决，让大脑专注于判断。

# 错误

  • 无知错误

不知道而产生的错误

  • 无能之错

不记得，忽略而产生的错误

# 清单分类

  • 执行清单

简单问题的解决方法，直接执行无需思考

  • 核查清单

解决在复杂情况下忽视问题的时候，解决无能错误

  • 沟通清单

多人协作，专业的事让专业的人做，记录需要与专人沟通后做的事
