## 思维导图

| 操作 | 快捷键 |
| --- | --- |
| 增加同级节点 | Enter |
| 增加子节点 | Tab |
| 添加链接/链接笔记 | Cmd/Ctrl + K |
| 移除链接 | Shift + Cmd + K (Mac) Shift + Ctrl + R (Windows) |
| 放大笔记 | Cmd/Ctrl + 「+」 |
| 缩小笔记 | Cmd/Ctrl + 「-」 |
| 笔记回到默认大小 | Cmd/Ctrl + 0 |
| 删除节点(选中节点时) | ⌫ |
| 撤销 | Cmd/Ctrl + Z |
| 重做 | Cmd/Ctrl + Shift + Z |
