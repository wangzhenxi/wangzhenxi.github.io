# Xmind

## 层叠关系

- 画布
 - 中心主题（有且仅有一个）
     + 分支主题
         + 小主题

## 快捷键

| 快捷键 | 描述 |
| --- | --- |
| Tab | 创建下一级主题 |
| Enter | 创建同级主题（向下） |
| Shift + Enter | 创建同级主题（向上） |
| Ctrl + C | 复制整个主题 |
| Ctrl + V | 在选中的主题下，将复制的主题添加至下一级主题 |
| Ctrl + + | 放大视图 |
| Ctrl + - | 缩小视图 |
| 右击 | 移动视图 |
| Ctrl + T | 新建画布 |
