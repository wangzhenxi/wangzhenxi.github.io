# visio

## 简介

一款用于绘制流程图的软件

## 快捷键

| 功能 | 按键 |
| --- | --- |
| 选择工具 | CTRL + 1 |
| 文字工具 | CTRL + 2 |
| 连接线 | CTRL + 3 |
| 形状工具 | CTRL + 8 |
| 文本加粗/斜体/删除线 | CTRL + B/I/U |
| 字体变大/小 | CTRL + Shift + ,/. |
| 组合/取消组合 | CTRL + G / CTRL + Shift + U |
| 所选元素置于顶/底部 | CTRL + Shift + F/B |
| 格式刷 | CTRL + Shift + P |
| 复制 | CTRL + 左键拖动 |
| 水平/垂直复制 | CTRL + Shift + 左键拖动 |
| 放大 | CTRL + 滚轮 |
| 水平滚动 | Shift + 滚轮 |
