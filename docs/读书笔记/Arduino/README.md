## 电阻

### 色环电阻

![](./image/2022-05-16-09-36-13.png)

通过在电阻上涂不同的色环，来区分电阻的阻值。常用的分为四环和五环电阻：

- 四环电阻： 前二环代表数字，第三环代表阻值倍数，最后一环为误差。
- 五环电阻： 前三环代表数字，第四环代表阻值倍数，最后一环为误差。
